Influences

First of all, this builds on the foundation of Dungeon World by Sage LaTorra and Adam Koebel, who did a great job in creating a Dungeons and Dragons style game using Vincent Baker's Powered by the Apocalypse rules.

Tools

Dungeon Chronicles was created in Eclipse.

The original text was written entirely in HTML with a CSS stylesheet. Working in plaintext allowed versioning the files using Git and Github.
